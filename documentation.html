How to use this service

REST Service

It is possible to use simple REST service in order to retrieve data from the server. REST service is available under the /rest/ adress. There are implemented three basic filtering options:

  • Date-based filtering
  • Depth-based filtering
  • Commit-based filtering

Each filter is documented in one of the following sections.

Date based filtering

/rest/date/<project_id>/<from_date>/<to_date>/
  • <project_id>
  • <from_date>
  • <to_date>
Example
/rest/date/<project_id</<from_date>/<to_date>/

Depth based filtering

/rest/depth/<project_id>/<depth>/<from_commit>/
  • <project_id>
  • <depth>
  • <from_commit>
Example
/rest/depth/<project_id>/<depth>/<from_commit>/

Commit based filtering

/rest/commit/<project_id>/<from_commit>/<to_commit>/
  • <project_id>
  • <from_commit>
  • <to_commit>
Example
/rest/depth/<project_id>/<depth>/<from_commit>/
